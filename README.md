# n8n-Uncle-Tiger

**Uncle-Tiger** เป็นผู้ช่วยบันทึกรายจ่ายอัตโนมัติ ทำงานผ่าน LINE Official Account

ผู้ใช้แค่ ถ่ายรูปสลิป แล้วส่งเข้าแชท ระบบจะอ่านสลิปให้เอง แยกว่าซื้อจากร้านไหน จ่ายไปกี่บาท วันที่เท่าไหร่ แล้วจัดหมวดหมู่ให้ พร้อมเก็บลงชีทบันทึกรายจ่าย จากนั้นตอบ
กลับเข้า LINE ทันทีว่า "บันทึกแล้วนะ" — ไม่ต้องเปิดแอปแยก ไม่ต้องพิมพ์เอง ไม่ต้องกรอกฟอร์ม
